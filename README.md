# Arbitrage-pricing-model
Excel-based financial model for futures pricing, arbitrage detection, and implied risk-free rate calculation using the Cost of Carry model.

# Arbitrage Pricing Model using Futures Contracts

## File Description
### Futures_Arbitrage_Analysis.xlsx
This Excel model contains:

#### Section 1: Futures Arbitrage Detection
- Underlying asset details
- Expiry date calculations
- Spot price input
- Risk-free rate assumption
- Theoretical futures price calculation
- Comparison with actual futures price
- Arbitrage profit identification
- Cash-and-carry arbitrage strategy explanation

#### Section 2: Implied Risk-Free Rate Calculation
- Spot and futures price inputs
- Time to expiry calculation
- Reverse engineering implied RFR
- Validation of futures pricing equation

## Key Learnings
- Practical implementation of derivatives pricing
- Relationship between spot and futures markets
- Arbitrage-free pricing mechanisms
- Financial modelling using Excel formulas
